# Tharun S Portfolio

A premium personal portfolio website built with:

- React
- Node.js
- Express.js
- MongoDB

## Run the frontend

```bash
cd frontend
npm install
npm start
```

## Run the backend

```bash
cd backend
npm install
npm start
```

## Features

- Responsive portfolio
- Dark/Light mode
- Projects
- Skills
- Coding Profiles
- Contact form with MongoDB and Nodemailer
